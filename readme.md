To run backtester use the following command, can specify round and days:

prosperity4btest trader.py <round>
prosperity4btest trader.py <round>-<day>

To Visualise put --vis at the end of the command:

prosperity4btest trader.py <round> --vis
prosperity4btest trader.py <round>-<day> --vis

You will then need to upload the coresponding log file to the website.

To not produce a log file put --no-out at the end of the command:

prosperity4btest trader.py <round> --no-out
prosperity4btest trader.py <round>-<day> --no-out


Example commands:

Visualise round 0 day -2: prosperity4btest trader.py 0--2 --vis
